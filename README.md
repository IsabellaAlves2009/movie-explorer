# 🎬 Movie API Project

Uma aplicação web para consulta de filmes, desenvolvida com React, TypeScript e estilizada com CSS Puro. O projeto consome uma API externa de cinema para listar filmes, exibir detalhes e permitir buscas em tempo real.

---

## 🚀 Tecnologias Utilizadas

* **React**
* **TypeScript**
* **CSS3**
* **Fetch API / Axios**

---

## ✨ Funcionalidades

* 🔍 **Busca de Filmes:** Encontre filmes por título através da barra de pesquisa.
* ⭐ **Destaques e Populares:** Listagem dos filmes do momento.
* 📄 **Página de Detalhes:** Informações detalhadas sobre o filme selecionado (sinopse, nota, data de lançamento).
* 📱 **Design Responsivo:** Interface adaptável para dispositivos móveis e computadores.
